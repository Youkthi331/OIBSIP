## What it is
A small Python project that calculates Body Mass Index (BMI) and displays the BMI category. Includes:
- CLI app (`main.py`)
- Tkinter GUI (`gui.py`)
- Reusable core logic (`bmi_module.py`)
- Unit tests (`tests.py`)
- Visual assets (`assets/`)

## Features
- Validates inputs (rejects non-numeric, zero, negative).
- CLI and GUI use the same core logic.
- Unit tests with pytest.
- Clear README and demo assets.
