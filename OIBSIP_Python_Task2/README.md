# Password Generator (Python)

A simple password generator built in Python using `random` and `string` libraries.  
Generates strong passwords based on user-selected length and character types.

## Objective
To create a beginner-friendly Python tool that can generate random, secure passwords with adjustable length.

## Features
- Generate random passwords
- Includes uppercase, lowercase, digits, and symbols
- Adjustable password length
- CLI version + GUI (Tkinter)
- Tests included

## Steps Performed
1. Wrote the core password generation logic in `password_module.py`
2. Built a command-line interface in `main.py`
3. Added a Tkinter GUI (`gui.py`)
4. Added unit tests (`tests.py`)
5. Collected demo screenshots + screen recording and saved inside `assets/`
6. Verified project execution using Windows CMD

## Tools Used
- Python 3  
- Modules: random, string, tkinter  
- VS Code  
- Windows CMD
